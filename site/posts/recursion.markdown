Theorem:
If $a$ is an element in set $X$, $f$ is a function from $X$ into $X$, and
$\omega$ is a successor set, then there is a function $u$ from $\omega$ to X
that $u(0)$ $=$ $a$, and for every $n$ in $\omega$, $u(n^+)$ $=$ $f(u(n))$.
