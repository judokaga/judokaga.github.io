Euler's Theorem

Posted on September 22, 2017 by Judokaga

Theorem (Euler):

If \(a\) is prime to \(m\), then \(a^{\phi(m)} \equiv 1 \mod m\).

Proof:

Let the set of numbers in \([1, m]\) that are prime to \(m\) be \(E\), we prove \(E\) is a group with multiplication modulo m being the group product.
Let \(e = 1\), for \(x \in E\), \(xe = ex = x\). If \(a, b \in E\), then there exists \(u_1, v_1\) that \(au_1 + mv_1 = 1\) and \(u_2, v_2\) that \(bu_2 + mv_2 = 1\), so \(u_1u_2(ab) + (au_1v_2 + bu_2v_1 + v1_v2m)m = 1\), so \((ab \bmod m, m) = 1\), so \(ab \bmod m\) is prime to \(m\), so \(ab \in E\). For any \(a \in E\), there exists \(u, v\) that \(ua + vm = 1\), that is \((ua \bmod m) = e\), so \(u \bmod m\) is \(a^{-1}\). So we proved \(E\) is a group.

\(E\) is a group, and \(o(E) = \phi(m)\), Let the cyclic subgroup of \(E\) generated by \(a\) be \(A\), then \(o(A) | \phi(m)\). \(a^{o(A)} = e \to a^{\phi(m)} = e \to a^{\phi(m)} \equiv 1 \mod m\).
